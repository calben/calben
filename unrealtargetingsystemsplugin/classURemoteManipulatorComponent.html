AActor * URemoteManipulatorComponent::StartManipulatingByRelativePosition ( )

Starts manipulating using the relative position controller.

The relative position controller is most intuitive in VR, but it can be used in any mode. A point in space is designated to be the origin of the manipulation. The delta between the manipulator and origin is applied to the object as a force.

Returns
